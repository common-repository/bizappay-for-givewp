Bizappay for GiveWP. 

== Description ==

This plugin enable online payment using online banking (FPX) and only available for online transactions that reside in Malaysia. This plugin was developed by a third party vendor (Laratech Systems) and is not under the supervision of Bizappay

== Installation ==

1. Make sure that you already have GiveWP plugin installed and activated.
2. From your Wordpress admin dashboard, go to menu 'Plugins' and 'Add New'.
3. Key in 'Bizappay' in the 'Search Plugins' field and press enter.
4. Select Bizappay for GiveWP and press intall.
5. Activate the plugin through the 'Plugins' screen in WordPress.
6. Go to menu Donations > Settings > Payment Gateways. Enable Bizappay and set ad Default and click Save Changes button.
8. On the same tab (Payment Gateways), select Bizappay and fill up required information which you can get from your Bizappay's account. (www.bizappay.my)

== Frequently Asked Questions ==

= Do I need to sign up with Bizappay (www.bizappay.my) in order to use this plugin? =

Yes, the plugin require info such as API key and category code that is only available after you sign up with Bizappay.

= Is there any specific charges for using this plugin? =

All the successful transactions via online banking will be charged MYR 1.00 (standard charge rate) by the payment gateway (Bizappay).

= What currency does it support? =

This plugin only support Malaysian Ringgit (MYR).
